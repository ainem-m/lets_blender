## 1. あなたのPCは大丈夫？Blenderの推奨スペックを確認

Blenderは高機能なソフトですが、意外にも一般的なPCで動作します。
しかし、快適に作業するためには、ある程度のPCスペックが求められます。
公式サイトで推奨されているスペックを確認してみましょう。

-   **[Blender公式サイト - 推奨スペック](https://www.blender.org/download/requirements/)**

**【最低限必要なスペック】**
*   **CPU:** 4コア・SSE4.2対応
*   **メモリ:** 8GB RAM
*   **グラフィックボード(GPU):** 2GB VRAM搭載のOpenGL 4.3対応カード
*   **ディスプレイ:** 1920x1080 (フルHD)

**【推奨スペック】**
*   **CPU:** 8コア
*   **メモリ:** 32GB RAM
*   **グラフィックボード(GPU):** 8GB VRAM搭載のNVIDIAまたはAMD製カード

**「自分のPCスペックが分からない！」という方へ**
-   **Windows:** `設定` > `システム` > `詳細情報` で確認できます。
-   **Mac:** `Appleメニュー` > `このMacについて` で確認できます。

もしスペックが足りなくても、Blenderの古いバージョンを使ったり、設定を軽くしたりすることで対応できる場合もあります。まずはインストールしてみましょう！

## 2. Blender本体をダウンロードしよう

早速Blenderをダウンロードします。

1.  **[Blender公式サイトのダウンロードページ](https://www.blender.org/download/)** にアクセスします。
    *   [Screenshot: Blender公式サイトのダウンロードページ]
2.  ページがあなたのOS（Windows/Mac）を自動で判別し、最適なバージョンを大きく表示してくれます。
3.  青い **「Download Blender [バージョン番号]」** ボタンをクリックしてください。ダウンロードが始まります。
    *   [Screenshot: ダウンロードボタンを指し示す画像]

**※もし違うOSのバージョンが必要な場合**
ボタンの下にあるドロップダ��ンメニューから、Windows (MSI installer), macOS (Apple Silicon / Intel), Linuxなどを選択できます。

## 3. OS別！Blenderインストール手順

ダウンロードしたファイルを使って、BlenderをPCにインストールします。
お使いのOSの項目に進んでください。

### Windowsの場合

Windowsでは、インストーラーを使って数回クリックするだけで完了します。

1.  ダウンロードした `.msi` ファイルをダブルクリックして実行します。
    *   [Screenshot: ダウンロードした.msiファイル]
2.  **"Welcome to the Blender Setup Wizard"** という画面が表示されたら、「**Next**」をクリックします。
    *   [Screenshot: セットアップウィザード開始画面]
3.  ライセンス契約の画面になります。**"I accept the terms in the License Agreement"** のチェックボックスにチェックを入れ、「**Next**」をクリックします。
    *   [Screenshot: ライセンス同意画面]
4.  インストール先を選択します。基本的には変更せず、そのまま「**Next**」をクリックして問題ありません。
    *   [Screenshot: インストール先選択画面]
5.  「**Install**」ボタンをクリックすると、インストールが開始されます。
    *   [Screenshot: インストール開始ボタン]
6.  ユーザーアカウント制御のポップアップが出たら「はい」を選択します。
7.  インストールが完了すると **"Completed the Blender Setup Wizard"** と表示されます。「**Finish**」をクリックしてウィンドウを閉じます。
    *   [Screenshot: インストール完了画面]

これでインストールは完了です！スタートメニューに「Blender」が追加されているので、確認してみましょう。

### macOSの場合

Macの場合は、ダウンロードしたファイルを「アプリケーション」フォルダに移動させるだけです。

1.  ダウンロードした `.dmg` ファイルをダブルクリックして開きます。
    *   [Screenshot: ダウンロードした.dmgファイル]
2.  Blenderのアイコンと、「Applications」フォルダのエイリアス（ショートカット）が表示されたウ���ンドウが開きます。
    *   [Screenshot: .dmgファイルを開いた後のウィンドウ]
3.  Blenderのアイコンを、マウスでドラッグして「Applications」フォルダの上に持っていき、ドロップします。
    *   [Screenshot: アイコンをドラッグ＆ドロップしている様子]
4.  ファイルのコピーが完了すれば、インストールは完了です。

Launchpadまたは「Applications」フォルダからBlenderを起動できます。

**【初回起動時の注意】**
Macでは、インターネットからダウンロードしたアプリを初めて開く際に警告が表示されることがあります。
その場合は、`システム設定` > `プライバシーとセキュリティ` で「このまま開く」を許可するか、Blenderアプリアイコンを右クリック（またはControlキーを押しながらクリック）して「開く」を選択してください。

### Linuxの場合

Linuxでは、ダウンロードしたファイルを解凍するだけで使えます。

1.  ダウンロードした `.tar.xz` ファイルを、好きな場所で右クリ��クし「ここに展開する」などを選択して解凍します。
2.  解凍されてできたフォルダの中に `blender` という名前の実行ファイルがあります。
    *   [Screenshot: 解凍後のフォルダと実行ファイル]
3.  この `blender` ファイルをダブルクリックすると、Blenderが起動します。

毎回このフォルダから起動するのが面倒な場合は、デスクトップへのショートカット（シンボリックリンク）を作成しておくと便利です。

## 4. 【重要】Blenderの初回起動と日本語化

インストールお疲れ様でした！早速Blenderを起動してみましょう。

1.  スタートメニューやApplicationsフォルダからBlenderを起動します。
2.  初回起動時のみ、スプラッシュスクリーン（設定画面）が表示されます。ここで基本的な設定を行います。
    *   [Screenshot: 初回起動時のスプラッシュスクリーン]

**ここで必ず日本語化しておきましょう！**

3.  **Language** の項目を「**Japanese (日本語)**」に変更します。
    *   [Screenshot: Language選択部分をハイライト]
4.  これだけで、メニューなどが日本語表示に切り替わります。
5.  他の設定は後からでも変更できるので、右上の「**設定を保存**」ボタンをクリックします。

これで、Blenderのインターフェースが日本語になりました！

## 5. 【最重要】GPU設定をしてレンダリングを高速化しよう

この設定は、Blenderを快適に使う上で非常に重要です。
PCにグラフィックボード（GPU）が搭載されている場合、GPUを使って画像の計算（レンダリング）を行うように設定すると、CPUで行うよりも劇的に速くなります。

1.  トップメニューの `編集` > `プリファレンス` を選択します。
    *   [Screenshot: 編集 > プリファレンス メニュー]
2.  開いたウィンドウの左側から `システム` タブを選択します。
    *   [Screenshot: プリファレンスウィンドウのシステムタブ]
3.  **サイクルレンダーデバイス** という項目を探します。ここに、お使いのPCに搭載され��いるGPUの種類に応じた選択肢が表示されます。
    *   **NVIDIA製GPU (GeForceなど) の場合:** 「**CUDA**」または「**OptiX**」を選択します。基本的には新しい「OptiX」がおすすめです。チェックボックスにチェックを入れると、その下にご自身のグラフィックボード名が表示されるので、それにもチェックを入れます。
        *   [Screenshot: OptiX選択画面]
    *   **AMD製GPU (Radeonなど) の場合:** 「**HIP**」を選択します。チェックボックスと、その下のグラフィックボード名にチェックを入れます。
        *   [Screenshot: HIP選択画面]
    *   **Intel製GPU (Arcなど) の場合:** 「**oneAPI**」を選択します。チェックボックスと、その下のグラフィックボード名にチェックを入れます。
        *   [Screenshot: oneAPI選択画面]

4.  設定が終わったら、左下の「**プリファレンスを保存**」を忘れずにクリックしてください。

これで、レンダリング時にGPUのパワーを最大限に活かせるようになりました！

## 6. つまずきやすいポイントとQ&A

**Q. インストールしたのにBlenderが起動しません！**
**A.** いくつか原因が考えられます。
*   **原因1: PCスペック不足**  
    お使いのPCが、Blenderの[最低動作スペック](https.www.blender.org/download/requirements/)を満たしているか確認してください。特にグラフィックボードが古いと起動しないことがあります。
*   **原因2: グラフィックドライバが古い**  
    GPUの性能を最大限に引き出すための「ドライバ」というソフトが古い可能性があります。NVIDIA, AMDなど、お使いのGPUメーカーの公式サイトから最新のドライバをダウンロードしてインストールしてみてください。
*   **原因3: (Windows) Visual C++ 再頒布可能パッケージがない**  
    少し専門的になりますが、Blenderの動作に必要なMicrosoftのプログラムがPCに入っていない場合があります。[こちらのページ](https://learn.microsoft.com/ja-jp/cpp/windows/latest-supported-vc-redist)から最新版をインストールしてみてください。

**Q. レンダリングがすごく遅いです…**
**A.** 上記の「**5. GPU設定**」が正しくできているか、もう一度確認してみてください。GPUではなくCPUで計算していると、非常に時間がかかってしまいます。

**Q. Macで「開発元が未確認のため開けません」と表示されます。**
**A.** これはMacのセキュリティ機能によるものです。Blenderは安全なソフトなのでご安心ください。
`システム設定` > `プライバシーとセキュリティ` を開き、セキュリティの項目で「このまま開く」を許可してください。または、Blenderのアイコンを右クリックして「開く」を選択すれば起動できます。
