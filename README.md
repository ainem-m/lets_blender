## まずは自分のPC��知ろう！Blenderの推奨スペックを確認

Blenderは高機能なソフトですが、意外にも一般的なPCで動作します。
しかし、快適に作業するためには、ある程度のPCスペックが求められます。
公式サイトで推奨されているスペックを確認してみましょう。

-   **[Blender公式サイト - 推奨スペック](https://www.blender.org/download/requirements/)**

**【最低限必要なスペック】**
*   **CPU[^4]:** 4コア・SSE4.2[^5]対応
*   **メモリ[^6]:** 8GB RAM
*   **グラフィックボード(GPU)[^7]:** 2GB VRAM[^8]搭載のOpenGL 4.3[^9]対応カード
*   **ディスプレイ:** 1920x1080 (フルHD[^10])

**【推奨スペック】**
*   **CPU:** 8コア
*   **メモリ:** 32GB RAM
*   **グラフィックボード(GPU):** 8GB VRAM搭載のNVIDIAまたはAMD製カード

**「自分のPCスペックが分からない！」という方へ**
-   **Windows:** `設定` > `システム` > `詳細情報` で確認できます。
-   **Mac:** `Appleメニュー` > `このMacについて` で確認できます。

もしスペックが足りなくて���、Blenderの古いバージョンを使ったり、設定を軽くしたりすることで対応できる場合もあります。まずは焦らず、インストールしてみましょう！

### 最新版とLTS版

Blenderは最新版か、LTS版（長期サポート版）を使用するとよいでしょう。

本書では、主にBlender 3.6 LTS で解説していますが、最新版でも大きく違いはありません。本書と同じ画面で作業したい方は、長期サポート版のダウンロードページ (https://www.blender.org/download/lts/) から、3.6 LTSをダウンロードして、インストールしてみましょう。

## 【最重要】GPU設定をしてレンダリングを高速化しよう

この設定は少し専門的に見えるかもしれませんが、Blenderをサクサク快適に使うための、とても重要なステップです。ここを乗り越えれば、後の作業が劇的に楽になりますよ！
PCにグラフィックボード（GPU）が搭載されている場合、GPUを使って画像の計算（レンダリング）を行うように設定する��、CPUで行うよりも劇的に速くなります。

1.  トップメニューの `編集` > `プリファレンス` を選択します。
    *   [Screenshot: 編集 > プリファレンス メニュー]
2.  開いたウィンドウの左側から `システム` タブを選択します。
    *   [Screenshot: プリファレンスウィンドウのシステムタブ]
3.  **サイクルレンダーデバイス[^16]** という項目を探します。ここに、お使いのPCに搭載されているGPUの種類に応じた選択肢が表示されます。
    *   **NVIDIA製GPU (GeForceなど) の場合:** 「**CUDA**」または「**OptiX**」[^17]を選択します。よく分からなければ、新しい「OptiX」がおすすめです。チェックボックスにチェックを入れると、その下にご自身のグラフィックボード名が表示されるので、それにもチェックを入れます。
        *   [Screenshot: OptiX選択画面]
    *   **AMD製GPU (Radeonなど) の場合:** 「**HIP**」[^17]を選択します。チェックボックスと、その下のグラフィックボード名にチェックを入れ��す。
        *   [Screenshot: HIP選択画面]
    *   **Intel製GPU (Arcなど) の場合:** 「**oneAPI**」[^17]を選択します。チェックボックスと、その下のグラフィックボード名にチェックを入れます。
        *   [Screenshot: oneAPI選択画面]

4.  設定が終わったら、左下の「**プリファレンスを保存**」を忘れずにクリックしてください。

これで、レンダリング時にGPUのパワーを最大限に活かせるようになりました！

# もしもトラブルが起きても大丈夫！よくある質問と解決策

インストールで困ったことが起きても、多くの場合は簡単なことで解決します。焦らず、ご自身の状況に近いものを探してみてください。

**Q. インストールしたのにBlenderが起動しません！**
**A.** いくつか原因が考えられます。
*   **原因1: PCスペック不足**  
    お使いのPCが、Blenderの[最低動作スペック](https://www.blender.org/download/requirements/)を満たしているか確認してください。特にグラフィックボードが古いと起動しないことがあります。
*   **原因2: グラフィックドライバ[^18]が古い**  
    GPUの性能を最大限に引き出すための「ドライバ」というソフトが古い可能性があります。NVIDIA, AMDなど、お使いのGPUメーカーの公式サイトから最新のドライバをダウンロードしてインストールしてみてください。
*   **原因3: (Windows) Visual C++ 再頒布可能パッケージ[^19]がない**  
    少し専門的になりますが、Blenderの動作に必要なMicrosoftのプログラムがPCに入っていない場合があります。[こちらのページ](https://learn.microsoft.com/ja-jp/cpp/windows/latest-supported-vc-redist)から最新版をインストールしてみてください。

**Q. レンダリングがすごく遅いです…**
**A.** 上記の「**GPU設定**」が正しくできているか、もう一度確認してみてください。GPUではなくCPUで計算していると、非常に時間がかかってしまいます。設定一つで大きく変わりますよ！

**Q. Macで「開発元が未確認のため開けません」と表示されます。**
**A.** これはMacのセキュリティ機能によるものです。Blenderは安全なソフトなのでご安心ください。
`システム設定` > `プライバシーとセキュリティ` を開き、セキュリティの項目で「このまま開く」を許可してください。または、Blenderのアイコンを右クリックして「開く」を選択すれば起動できます。

# ショートカットキーについて 

Blenderはショートカットキーを覚えることで、作業効率が格段に上がります。本書で は、ショートカットキーを積極的に使って解説しています。

ショートカットキーは、キーボードのキーを押すことで、機能を呼び出すことができる便利な機能です。例えば、オブジェクトを移動させたいときは、Gキーを押します。

本書では、ショートカットキーを次のように表記しています。

-   G → 移動
-   S �� 拡大・縮小
-   R → 回転
-   X → X軸に固定
-   Y → Y軸に固定
-   Z → Z軸に固定
-   Shift + A → 追加メニュー
-   Ctrl + Z → 元に戻す
-   Tab → 編集モードとオブジェクトモードの切り替え

これらのショートカットキーは、Blenderの基本的な操作でよく使うものです。最初は覚えるのが大変かもしれませんが、使っていくうちに自然と指が覚えていきます。

[^12]: **.dmg:** macOSで使われるディスクイメージファイル。ソフトウェアを配布する際によく使われる形式。
[^13]: **エイリアス:** Windowsの「ショートカット」と同じ機能。別の場所にあるファイルやフォルダを、あたかもそこにあ���かのように見せかけるための分身アイコン。
[^14]: **スプラッシュスクリーン:** ソフトウェアの起動時に一瞬表示される、ロゴやバージョン情報が書かれた画面。Blenderでは初回起動時に簡単な初期設定画面を兼ねる。
[^15]: **インターフェース:** 利用者がソフトウェアを操作するための画面デザインやメニュー、ボタンなどの総称。
